# Automatizacion-de-instalacion-de-servicios-mediante-Ansible
Realizado en entorno debian, con una lista básica de servicios y contiene ficheros para poder llevar a cabo la monitorización de las automatizaciones con Ansible

Lista de servicios:
  - DNS (bind9)
  - DHCP (isc-dhc-server)
  - Samba
  - Apache
  - Docker
  - FTP (vsftpd)
  - DDNS (bind9 + isc-dhcp-server)
  - Pila LAMP (Apache + Mysql + PHP) [Incluído libapache2-mod-php y php-mysql]
